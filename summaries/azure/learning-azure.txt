
==What is Azure?

Cloud computing platform offering a set of hosting services:

--Web
--VMs
--Network
--Storage
--Database
--Centralized account management
--AI
--ML
--IoT

Services are delivered over the internet through datacenters run by cloud providers

--Basic services: compute power and storage

--A pay-as-you-go pricing model for the cloud resources that you use

--Virtualization

Using a hypervisor
